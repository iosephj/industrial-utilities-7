Este repositorio tiene todos los contenidos de la materia "Instalaciones Industriales" de 7mo E (electrónica). Está estructurado para ponerlos en un LMS como Canvas, ya sea a través de links web o de archivos qti para quizes.

Si el LMS es Canvas hay que tener en cuenta que:

- en Canvas conviene mantener todo (lecciones, actividades, etc.) en una secuencia lineal (módulos), es decir no no separar por tipo.

- en el repositorio, en cambio, conviene separar por tipo (un poco) para poder mantener y automatizar.

Es decir Canvas y el Repositorio no tienen por qué tener la misma estructura. Esta es una idea fuerte (la más importante): Separar “cómo se guarda” de “cómo se muestra”

- Repo → optimizado para la escalabilidad y el mantenimiento
- Canvas → optimizado para alumno (flujo lineal)
